Independent set problem

Problem definition

An independent set is defined in the monadic second order language as

\[\exists x_i,\ldots,x_M\left[\wedge_{i\neq j} (x_i\neq x_j \wedge \neg \textbf{adj}(x_i, x_j))\right]\]
